Part 1: CSS Styling Guide for "LegalNations"
Here are the CSS specifications based on the typeface and color hierarchy seen in the presentation mockup. This guide uses widely available web-safe/Google Fonts to achieve a similar look.

1. Core Variables & Colors
First, define your brand colors. I’ve extracted the key hexadecimal values from the mockup.

CSS
:root {
  /* Brand Blue - Used for 'Nations' and icons */
  --legalnations-blue: #0056d2;
  /* Deep Charcoal/Black - Used for 'Legal' and main text */
  --legalnations-dark: #333333;
  /* Light Gray - Used for subtitles/body text */
  --legalnations-gray: #666666;
}
2. Typography & Sizing Hierarchy
To replicate the professional, geometric sans-serif font, Inter (available via Google Fonts) is a great choice that matches the weight and clarity.

Google Font Embed:
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">

CSS
body {
  font-family: 'Inter', sans-serif;
  color: var(--legalnations-dark);
}

/* --- Main Brand Wordmark (LegalNations) --- */
.brand-wordmark {
  font-weight: 700; /* Bold */
  letter-spacing: -0.02em; /* Tight kerning */
  font-size: 48px; /* Adjust based on your layout needs */
  line-height: 1.2;
}

.brand-wordmark .legal {
  color: var(--legalnations-dark);
}

.brand-wordmark .nations {
  color: var(--legalnations-blue);
}

/* --- Document/Presentation Subtitles ("Proposed Logo 1...") --- */
.presentation-subtitle {
  font-weight: 400; /* Regular */
  font-size: 14px;
  color: var(--legalnations-gray);
  text-transform: uppercase; /* Match the title style */
  letter-spacing: 0.1em; /* Increased letter spacing */
}

/* --- Large Web Headlines ("Register Your Company Anywhere...") --- */
.hero-headline {
  font-weight: 700; /* Bold */
  font-size: 36px;
  line-height: 1.3;
  color: #ffffff; /* Typically used on dark backgrounds, like the dashboard */
}
3. HTML Example
Here is how you would structure the main brand wordmark to apply the two-tone effect.

HTML
<div class="brand-wordmark">
  <span class="legal">Legal</span><span class="nations">Nations</span>
</div>
